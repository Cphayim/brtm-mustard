# brtm-mustard

## 记特喵的账服务端

基于 Flask 的 RESTFulAPI 服务


### 项目结构

```shell
.
├── Pipfile
├── Pipfile.lock
├── README.md
├── app     # 源码目录
│   ├── config          # 配置
│   ├── controllers     # 控制器（视图函数）
│   ├── enums           # 枚举
│   ├── exceptions      # 异常
│   ├── libs            # 库
│   ├── models          # 数据模型
│   ├── validators      # 验证器
│   └── view_models     # 视图模型
├── docs    # 文档
├── main.py # 入口模块
├── scripts # sh 脚本
└── tests   # 测试
```
